# Camada: Model
Esta pasta contém os "moldes" do sistema. Não há lógica de cálculo aqui, apenas a estrutura dos dados.

### Componentes:
- **Pet.java**: Atributos como nome, idade, peso, raça e uma referência ao objeto Endereco.
- **Endereco.java**: Classe auxiliar com os campos Rua, Número e Cidade.
- **Enums**:
    - `TipoAnimal`: (CACHORRO, GATO).
    - `Sexo`: (MACHO, FEMEA).
