# Camada: Util
Ferramentas de apoio que podem ser usadas por qualquer outra camada.

### Componentes:
- **Constantes**: Armazena a String `NAO_INFORMADO`.
- **FormatadorData**: Lógica para gerar o prefixo `20231101T1234`.
- **Normalizador**: Função para remover acentos e facilitar a busca case-insensitive.
